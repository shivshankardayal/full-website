    \[\phi_n(x + y) = \phi_n(x) + c^n_1\phi_{n - 1}(x)\phi_1(y) + c^n_2\phi_{n - 2}(x)\phi_2(y) + ... + C^n_{n - 1}\phi_1(x)\phi_{n - 1}(y) + \phi_n(y).\]

  7. Prove the following where \(p = x + y\) and \(q = xy\)

    1. \[x^n + y^n = p^n - np^{n - 2}q + \frac{n(n - 3)}{1*2}p^{n - 4}q^2 - ... +\]\[(-1)^r\frac{(n - r - 1)(n - r - 2) ... (n - 2r +1)}{r!}p^{n - 2r}q^r + ....\]
    2. \[\frac{x^{n + 1} - y^{n + 1}}{x - y} = p^n - C^{n - 1}_1p^{n - 2}q + ...+\]\[(-1)^rC^{n - r}_rp^{n - 2r}q^r + ...\]

  8. Let \(x + y = 1\) then prove that

    \[x^m(1 + C^m_1y + C^{m + 1}_2y^2 + ... + C^{2m -2}_{m - 2}y^{m - 1}) + ...\]\[+ y^m(1 + C^m_1x + C^{m + 1}_2x^2 + ... + C^{2m -2}_{m - 2}x^{m - 1}) = 1\]

  9. Prove that

    \[\frac{1}{(x - a)^m(x - b)^m} = \frac{1}{(a - b)^m}\left\{\frac{1}{(x - a)^m} + \frac{C^m_1}{(x - a)^{m - 1}(b - a)} + ... + \frac{C^{2m - 2}_{m - 1}}{(x - a)(b - a)^{m - 1}}\right\}\]\[+ \frac{1}{(b - a)^m}\left\{\frac{1}{(x - b)^m} + \frac{C^m_1}{(x -b)^{m - 1}(a - b)} + ... + \frac{C^{2m - 2}_{m - 1}}{(x - b)(a - b)^{m - 1}}\right\}.\]

  10. Show that constants \(A_1, A_2, A-3\) can always be chosen so that

    \[(x + y)^n = x^n + y^n + A_1xy(x^{n - 2} + y^{n - 2}) + A_2x^2y^2(x^{n -4}y^{n - 4}) + ...\]

  11. Let \(m\) and \(n\) be positive integers (\(n \le m\)). Let

    \[\frac{(1 - x^m)(1 - x^{m -1}) ... (1 - x^{m - n +1})}{(1 - x)(1 - x^2) ... (1 - x^n)} = (m, n).\]

    Prove that

    1. \((m, n) = (m, m - n),\)
    2. \((m , n + 1) = (m - 1, n + 1) + x^{m - n - 1}(m - 1, n),\)
    3. \((m, n + 1) = (n, n) + x(n + 1, n) + x^2(n + 2, n) + ... + x^{m - n - 1}(m - 1, n),\)
    4. \((m, n)\) is a polynomial in \(x\),
    5. \(1 - (m, 1) + (m, 2) - (m, 3) + ...\) is equal to \((1 - x)(1 -x^2) ... (1 - x^{m - 1})\) if \(m\) is even and 0 if \(m\) is odd.

  12. Prove that

    \[(1 + xz)(1 + x^2z) ... (1 + x^nz) = 1 +\]\[\sum_{k = 1}^{k = n}\frac{(1 - x^n)(1 - x^{n - 1}) ... (1 - x^{n - k +1})}{(1 -x)(1 - x^2) ... (1 -x^k)}x^\frac{k(k + 1)}{2}z^k.\]

  13. Prove that

    \[(1 + xz)(1 + x^2z) ... (1 + x^{2n -1}z) = 1 +\]\[\sum_{k = 1}^{k = n}\frac{(1 - x^{2n})(1 - x{2n - 2}) ... (1 - x^{2n - 2k + 2})}{(1 - x^2)(1 - x^4) ... (1 - x^2k)}x^{k^2}z^k.\]

  14. Let \(p_k = (1 - x)(1 - x^2) ... (1 - x^k).\) Prove that

    \[\frac{1}{p_n} - \frac{x}{p_1p_{n - 1}} + \frac{x^3}{p_2p_{n - 2}} - ... \pm \frac{x^{\frac{n(n + 1)}{2}}}{p_n} = 1.\]

  15. Determine the coefficients \(C_0, C_1, C-2, ..., C-n\) in the following identity

    \[(1 + xz)(1 + xz^{-1})(1 + x^3z)(1 + x^3z^{-1}) ... + (1 + x^{2n + 1}z)(1 + x^{2n + 1}z^{-1}) =\]\[C_0 + C_1(z + z^{-1}) + C_2(z^2 + z^{-2}) + ... + C_n(z^n + z^{-n}).\]

  16. Let

    \[u_k = \frac{sin 2nx sin (2n - 1)x ... sin (2n - k + 1)x}{sin x sin 2x ... sin kx}.\]

    Prove that

    \[1 - u_1 + u_2 - u_3 + ... + u_{2n} = 2^n(1 - cos x)(1 - cos 3x) ... [1 - cos (2n - 1)x]\]

    and also that

    \[1 - u_1^2 + u^2_2 - u^2_3 + ... + u_{2n}^2 = (-1)^n\frac{sin(2n + 2)x sin(2n + 4)x ... sin4nx}{sin2x sin4x ... sin2nx}.\]
